# Tutoriais e-SUS APS

Uma aplicação web interativa desenvolvida para facilitar o treinamento e a consulta sobre as principais funcionalidades do sistema **e-SUS APS** (Atenção Primária à Saúde). 

Este projeto visa auxiliar profissionais de saúde na assimilação e uso correto dos fluxos de trabalho do prontuário eletrônico.

## 🚀 Módulos de Treinamento

A aplicação oferece guias interativos passo a passo, links para manuais e acesso a vídeos de capacitação oficiais nos seguintes módulos:

*   **Gestão de Atendimentos**: Fluxo completo desde a Lista de Espera, Escuta Inicial, Vacinação, uso do método clinico SOAP até a Finalização.
*   **Unificação de Cadastros**: Passo a passo para resolver multiplicidade de cadastros do cidadão, unindo registros de forma segura no PEC e CDS.
*   **Gestão da Agenda**: Orientações sobre configuração da agenda, agendamento de consultas e reserva de horários para a equipe.
*   **Cadastros da APS**: Guias sobre o uso prático (via e-SUS Território) para Cadastro de Imóvel, Família e Cidadão Individual.
*   **Acompanhamentos**: Monitoramento do território através de listas temáticas (Gestantes, Pessoa Idosa, Hipertensão, Diabetes, etc.) e filtros avançados.

## 🛠️ Tecnologias Utilizadas

Este projeto front-end moderno foi construído utilizando:

*   **React 19** com **TypeScript**
*   **Vite** como bundler e servidor de desenvolvimento
*   **Tailwind CSS** para estilização rápida e responsiva
*   **Framer Motion** para transições de tela e animações
*   **Lucide React** para iconografia

## 🖥️ Como Executar Localmente

**Pré-requisitos:** Node.js instalado no seu sistema.

1. Instale as dependências do projeto:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. O Vite abrirá automaticamente ou fornecerá um link local (Ex: `http://localhost:5173`) para acessar a aplicação.
